## 2. 경쟁사 별 심층 분석

### 2-1. 3Commas

**개요**: 2017년 에스토니아 설립. 암호화폐 자동매매 플랫폼의 선두주자.

**가격 구조 (2026년 실제 확인)**:

- **Free**: AI 어시스턴트, 포트폴리오 트래커, DCA봇 1개, SmartTrade
- **Starter**: $15/월 (정가 $20, 연간 결제 시 25% 할인) — 시그널봇, 백테스팅(1년), 다중 봇, 1개 API 키
- **Pro**: $40/월 (정가 $50) — 선물봇, 3개 API 키, 고급 분석, 백테스팅(2년)
- **Expert**: $110/월 (정가 $140) — 무제한 봇, 우선 지원, 백테스팅(무제한)

**강점**:

- 20+ 거래소 연동 (Binance, Coinbase, Kraken, Bybit 등)
- 직관적인 UI/UX, 비개발자도 사용 가능
- 다양한 봇 유형 (DCA, Grid, Signal, Futures)
- 연간 결제 시 약 25% 할인

**약점**:

- 암호화폐 전용 → 한국 주식 시장 완전 미지원
- 2022년 해킹 사건으로 신뢰도 타격 (API 키 유출)
- 월 $110 Expert 플랜은 소규모 투자자에게 부담

**사용자 리뷰 요약**: 직관적 UI 호평, 봇 설정 복잡성 불만, 고객지원 느림, 연간 결제 자동 갱신 불만

**벤치마크**: $20M ARR, 83명 직원, 2017년 창업 → 8년 만에 달성

### 2-2. QuantConnect

**개요**: 2012년 설립. 기관급 알고리즘 트레이딩 플랫폼. LEAN 오픈소스 엔진 기반.

**가격 구조 (2026년 실제 확인)**:

- **Free**: 무제한 백테스팅, 주식/외환/암호화폐 데이터(분/시/일 단위), 커뮤니티 지원
- **Quant Researcher**: 유료 (VSCode/CLI 로컬 코딩, 틱/초 단위 데이터, 컴퓨트 노드 2개)
- **Team**: 팀 협업 (최대 10명), 컴퓨트 노드 10개
- **Trading Firm**: 무제한 팀, 권한 관리
- **Institution**: 온프레미스, AES-256 암호화, FIX 브로커

**강점**:

- LEAN 오픈소스 엔진 (GitHub 16,000+ stars, 180+ 기여자)
- 400TB+ 역사 데이터, 40+ 데이터 벤더
- 20+ 브로커 연동 (Interactive Brokers, Alpaca 등)
- 기관급 백테스팅 (10년 백테스트 33초)
- 300,000+ 사용자, $45B/월 거래량

**약점**:

- Python/C# 코딩 필수 → 비개발자 진입 완전 불가
- 한국 주식 시장 미지원 (KIS, 키움 브로커 없음)
- 복잡한 학습 곡선 (Bootcamp 필요)
- 한국어 지원 없음

**벤치마크**: $4M ARR, 23명 직원, 14년 운영

### 2-3. Pionex

**개요**: 2019년 설립. 거래소 + 자동매매 봇 통합 플랫폼.

**가격 구조**:

- 완전 무료 (16개 봇 포함)
- 수익 모델: 거래 수수료 0.05% (업계 최저)
- Binance + HTX 유동성 집계

**강점**:

- 무료 16개 봇 (그리드, DCA, 아비트라지, 리밸런싱 등)
- 업계 최저 수수료 0.05%
- API 키 불필요 (내장 거래소)
- 모바일 앱 우수 (App Store 4.3/5)

**약점**:

- 암호화폐 전용
- 규제 리스크 (FinCEN MSB만 보유, SEC 등록 없음)
- Trustpilot 2.3/5 (출금 불만 다수)
- 고객지원 이메일/채팅만 (전화 없음)

### 2-4. 이루다투자 (든든, 업라이즈투자자문)

**개요**: 국내 로보어드바이저 서비스. 2024년 11월 불리오(두물머리)와 전략적 결합 발표, 합산 AUM 5,000억원.

**가격 구조**:

- 연 0.15~0.5% 운용 수수료 (AUM 기반)
- 퇴직연금 일임 서비스 포함
- AUM 1,000만원 기준: 연 1.5~5만원

**강점**:

- 금융위원회 인가 로보어드바이저
- 퇴직연금(DC형) 자동 운용 가능 (2024년 하반기 허용)
- 국내 규제 완전 준수
- 결합 후 AUM 5,000억원 (업계 1위)

**약점**:

- ETF 중심 자산배분만 가능 (개별 주식 자동매매 불가)
- 전략 커스터마이징 완전 불가 (블랙박스)
- 활성 고객 1만명 수준 (성장 정체)
- 로보어드바이저 시장 전체 AUM 감소 추세

### 2-5. 불리오 (두물머리투자자문)

**개요**: 국내 AI 포트폴리오 서비스. 공모주 청약 특화.

**가격 구조**:

- 성과보수 모델 (수익 발생 시에만 수수료)
- 기본 운용 수수료 연 0.3~0.5%

**강점**:

- 공모주 청약 자동화 특화 (차별화 포인트)
- 성과보수로 심리적 진입장벽 낮음
- AUM 2,000억, 활성 1.3만명

**약점**:

- 개별 주식 자동매매 불가
- 전략 커스터마이징 불가
- 이루다투자와 합병 후 방향성 불확실

---

### 2-6. 키움증권 영웅문4 OpenAPI

**개요**: 국내 최대 개인투자자 증권사 키움증권의 공식 개발자 API.

**가격 구조**:

- 완전 무료 (API 자체)
- 키움증권 계좌 개설 필요
- 거래 수수료: 온라인 주식 0.015% (업계 최저 수준)

**강점**:

- 한국 주식 완전 지원 (코스피/코스닥/선물/옵션)
- 실시간 시세, 주문, 계좌 조회 모두 무료
- 방대한 커뮤니티 (네이버 블로그, 유튜브 튜토리얼 다수)
- 공식 API → 안정성 높음

**약점**:

- **Windows COM 전용** → macOS/Linux 불가
- **HTS(영웅문4) 상시 실행 필요** → 서버 운영 복잡
- Python 코딩 필수 (32bit Python 권장, 환경 설정 복잡)
- 백테스팅 기능 없음 (직접 구현 필요)
- 노코드 UI 없음 → 비개발자 완전 배제
- VPS 서버 직접 운영 필요 (월 3~10만원 추가 비용)

### LTV 계산

**Basic 플랜 LTV**:

```
월 9,900원 × 20개월 (1/5% 이탈률 기준) = 198,000원
연간 결제 비율 30% 가정 시 블렌딩: ~210,000원
LTV:CAC = 210,000 / 50,000 = 4.2:1 ✅ (목표 3:1 이상)
```

**Pro 플랜 LTV**:

```
월 29,900원 × 33개월 (1/3% 이탈률 기준) = 986,700원
연간 결제 비율 40% 가정 시 블렌딩: ~1,050,000원
LTV:CAC = 1,050,000 / 50,000 = 21:1 ✅ (우수)
```
