### JD 1: 풀스택 개발자 (TypeScript/Next.js)

**회사 소개**
AutoTrade KR은 코딩 없이 한국 주식 자동매매 전략을 만들고 실행할 수 있는 SaaS 플랫폼입니다. 개인 투자자가 감정 없이 규칙 기반 투자를 할 수 있도록 돕습니다.

**포지션**
풀스택 개발자 (프리랜서/파트타임)

**주요 업무**

- Next.js 기반 프론트엔드 기능 개발 및 개선
- Node.js/TypeScript 백엔드 API 개발
- 한국투자증권(KIS) API 연동 유지보수
- 백테스팅 엔진 성능 최적화
- 코드 리뷰 및 기술 문서 작성

**자격 요건**

- TypeScript 실무 경험 2년 이상
- Next.js 프로젝트 경험
- REST API 설계 및 개발 경험
- Git 기반 협업 경험

**우대 사항**

- 금융 도메인 또는 핀테크 경험
- PostgreSQL, Redis 경험
- AWS 배포 경험
- 주식 투자 경험 (실전 이해도)

**근무 조건**

- 형태: 프리랜서 계약 (주 20–30시간)
- 보수: 월 200–300만원 (경력에 따라 협의)
- 근무: 원격 근무 (주 1회 화상 미팅)

**지원 방법**
GitHub 프로필 + 포트폴리오 링크와 함께 이메일 지원

### JD 2: 콘텐츠 마케터 (금융 콘텐츠)

**회사 소개**
AutoTrade KR은 한국 주식 자동매매 SaaS 플랫폼으로, 개인 투자자가 코딩 없이 자신만의 투자 전략을 자동화할 수 있도록 돕습니다.

**포지션**
콘텐츠 마케터

**주요 업무**

- 주식 자동매매 관련 블로그 콘텐츠 기획 및 작성 (주 2–3편)
- 유튜브/숏폼 콘텐츠 기획 및 제작 지원
- 네이버 카페, 주식 커뮤니티 채널 관리
- 이메일 뉴스레터 기획 및 발송
- SEO 최적화 및 키워드 전략 수립
- 인플루언서/파트너십 협업 관리

**자격 요건**

- 금융/투자 관련 콘텐츠 제작 경험 1년 이상
- SNS 채널 운영 경험 (블로그, 유튜브, 인스타그램 중 1개 이상)
- 글쓰기 능력 (복잡한 개념을 쉽게 설명하는 능력)
- 데이터 기반 콘텐츠 성과 분석 경험

**우대 사항**

- 주식/ETF 투자 실전 경험
- 핀테크 또는 SaaS 마케팅 경험
- 영상 편집 능력 (프리미어, 캡컷 등)
- 네이버 블로그 상위 노출 경험

**근무 조건**

- 형태: 정규직 또는 프리랜서 협의
- 보수: 월 250–350만원
- 근무: 원격 근무 가능

**지원 방법**
포트폴리오(콘텐츠 샘플 3개 이상) + 이력서 이메일 지원

### JD 3: 고객지원 담당자

**회사 소개**
AutoTrade KR은 한국 주식 자동매매 플랫폼으로, 개인 투자자가 자신만의 투자 전략을 자동화할 수 있도록 돕는 SaaS 서비스입니다.

**포지션**
고객지원 담당자 (CS)

**주요 업무**

- 신규 사용자 온보딩 지원 (채팅, 이메일)
- 고객 문의 응대 및 문제 해결
- FAQ 및 헬프센터 문서 작성/업데이트
- 사용자 피드백 수집 및 개발팀 전달
- 이탈 사용자 재활성화 캠페인 지원

**자격 요건**

- 고객 응대 경험 1년 이상
- 주식 투자 기본 지식 (계좌 개설, 매수/매도 경험)
- 친절하고 명확한 커뮤니케이션 능력
- 기본적인 IT 서비스 이해도

**우대 사항**

- 자동매매 또는 알고리즘 트레이딩 경험
- SaaS 서비스 CS 경험
- 카카오톡 채널, 채널톡 등 CS 툴 사용 경험
- 콘텐츠 작성 능력 (가이드 문서 작성)

**근무 조건**

- 형태: 정규직
- 보수: 월 200–250만원
- 근무: 원격 근무 (응답 시간 오전 9시–오후 6시)

**지원 방법**
이력서 + 지원 동기 (500자 이내) 이메일 지원
