# AutoTrade KR — 법인 설립 체크리스트

> 주식회사 설립 절차 및 사업자 등록 가이드

---

## ⚠️ 중요 원칙

- **유료 서비스 운영 전 법인 설립 필수**
- **금융 관련 서비스이므로 법무 자문 강력 권장**
- **자금 소진 전 선제적으로 준비 시작**

---

## 1단계: 사전 준비

### 상호 결정

- [ ] 사용할 회사명 결정 (예: "오토트레이드코리아 주식회사")
- [ ] 특허청 상표 검색 (https://www.kipris.or.kr)
  - 동일/유사 상표 존재 여부 확인
  - 상표 등록 출원 검토 (선택사항, 비용 약 62,000원)
- [ ] 법인 등기 상호 중복 확인 (대법원 인터넷 등기소)
  - https://www.iros.go.kr → 상호 검색

### 정관 작성

- [ ] 정관 초안 작성
  - **목적 사업**: 소프트웨어 개발 및 공급업, 전자상거래, 정보통신업
  - **본점 소재지**: 실제 사무소 주소 (자택 가능)
  - **자본금**: 최소 100만원, 권장 1,000만원
  - **발행 주식 총수**: 10,000주 (액면가 100원 기준)
  - **임원**: 대표이사 1인 (1인 주주 가능)
- [ ] 법무사 또는 변호사 검토 (권장)

---

## 2단계: 자본금 납입

- [ ] 개인 통장에 자본금 입금 확인
  - 최소: 100만원
  - **권장: 1,000만원** (초기 운영비 포함)
- [ ] 잔액 증명서 발급 (은행 방문 또는 인터넷뱅킹)
  - 법인 설립 등기 시 제출 서류

---

## 3단계: 법인 설립 등기

### 방법 선택

| 방법             | 비용       | 소요 기간 | 난이도 |
| ---------------- | ---------- | --------- | ------ |
| 법무사 의뢰      | 50–100만원 | 3–5일     | 쉬움   |
| 온라인 직접 신청 | 10–20만원  | 5–7일     | 보통   |
| 법원 등기소 직접 | 10–20만원  | 5–7일     | 어려움 |

**권장**: 법무사 의뢰 (시간 절약, 오류 방지)

### 필요 서류

- [ ] 정관 (공증 필요 — 자본금 10억 미만 시 공증 생략 가능)
- [ ] 발기인 총회 의사록
- [ ] 이사회 의사록 (이사 3인 이상 시)
- [ ] 취임 승낙서 (대표이사)
- [ ] 인감증명서 (대표이사 개인)
- [ ] 주민등록등본
- [ ] 잔액 증명서 (자본금 납입 증명)
- [ ] 법인 인감 도장 (등기 전 제작)

### 등기 비용 (등록면허세)

```
자본금 1,000만원 기준:
- 등록면허세: 1,000만원 × 0.4% = 40,000원
  (최소 112,500원 적용)
- 지방교육세: 등록면허세 × 20% = 22,500원
- 등기신청 수수료: 20,000원
- 법무사 수수료: 300,000–700,000원
- 합계: 약 50–100만원
```

- [ ] 법원 등기소 방문 또는 온라인 등기 신청
  - 온라인: https://www.iros.go.kr (인터넷 등기소)
- [ ] 법인 등기부 등본 발급 확인

## 4단계: 사업자 등록

- [ ] 관할 세무서 방문 또는 홈택스 온라인 신청
  - 홈택스: https://www.hometax.go.kr

| 구분       | 업종명                    | 업종 코드 |
| ---------- | ------------------------- | --------- |
| **주업종** | 소프트웨어 개발 및 공급업 | 722000    |
| **부업종** | 전자상거래 소매업         | 479900    |

### 필요 서류

- [ ] 법인 등기부 등본 (발급일 3개월 이내)
- [ ] 정관 사본
- [ ] 대표이사 신분증
- [ ] 사업장 임대차 계약서 (자택 사용 시 자가 확인서)
- [ ] 법인 인감 도장

- [ ] 사업자 등록증 수령 (신청 후 3–5일)

## 5단계: 법인 통장 개설

- [ ] 은행 방문 (기업 전담 창구)
  - 추천 은행: 기업은행, 국민은행, 신한은행
- [ ] 필요 서류:
  - [ ] 법인 등기부 등본
  - [ ] 사업자 등록증
  - [ ] 법인 인감 도장
  - [ ] 대표이사 신분증
  - [ ] 법인 인감증명서
- [ ] 법인 공인인증서 발급 (인터넷뱅킹용)
- [ ] 법인 신용카드 발급 (경비 처리용)

## 6단계: 추가 행정 처리

### 세금 관련

- [ ] 부가가치세 일반과세자 등록 확인
  - 연 매출 8,000만원 미만: 간이과세 가능 (선택)
  - SaaS 서비스: 일반과세 권장
- [ ] 세무사 계약 (권장)
  - 월 기장료: 10–20만원
  - 부가세 신고, 법인세 신고 대행

### 4대보험 (직원 채용 시)

- [ ] 국민연금 사업장 가입 신고
- [ ] 건강보험 사업장 가입 신고
- [ ] 고용보험 사업장 가입 신고
- [ ] 산재보험 사업장 가입 신고
  - 신고처: 국민건강보험공단, 근로복지공단

### 기타

- [ ] 법인 이메일 도메인 설정 (예: contact@autotrade.kr)
- [ ] 법인 명함 제작
- [ ] 회계 소프트웨어 도입 (더존, 세금계산서 발행용)

## 법적 주의사항

### AutoTrade KR 특화 사항

**금융 서비스 관련**

- [ ] "비자문형 거래자동화 도구" 포지셔닝 법무 검토
- [ ] 서비스 이용약관 법무 검토 (투자 손실 면책 조항)
- [ ] 개인정보처리방침 작성 (개인정보보호법 준수)
- [ ] 전자금융거래법 적용 여부 확인

**자본시장법 관련**

- [ ] 투자자문업 해당 여부 법무 확인
  - 현재 포지셔닝: 해당 없음 (도구 제공)
  - 정기적 법무 모니터링 필요
- [ ] 금융위원회 혁신금융서비스 신청 검토 (선택)

**데이터 관련**

- [ ] 증권사 API 이용약관 준수 확인
- [ ] 사용자 투자 데이터 보관 정책 수립
