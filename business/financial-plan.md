## 매출 상세 분석

### ARPU 계산

```
Basic 플랜 (70%): 9,900원 × 0.7 = 6,930원
Pro 플랜 (30%):  29,900원 × 0.3 = 8,970원
ARPU = 6,930 + 8,970 = 15,900원 ≈ 15,000원 (보수적 적용)
```

### 월별 매출 계산 방식

```
월 매출 = 유료 사용자 수 × ARPU
예) Month 6: 155명 × 15,000원 = 2,325,000원 ≈ 230만원
```

### 유료 사용자 성장 모델

```
신규 유료 사용자 = 신규 무료 가입자 × 전환율 (10–15%)
이탈 사용자 = 전월 유료 사용자 × 이탈률 (5%)
순 증가 = 신규 유료 - 이탈

Month 1: 0명 (베타 기간)
Month 2: +10명 (초기 유료 전환)
Month 3: +20명 (베타 → 정식 출시)
Month 4–6: 월 +25–30명 (광고 시작)
Month 7–9: 월 +60–80명 (마케터 합류)
Month 10–12: 월 +100–130명 (성장 가속)
```
