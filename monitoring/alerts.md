## Sentry 설정

1. Sentry 프로젝트 생성 (Next.js)
2. SENTRY_DSN 환경변수 설정
3. 알림 규칙: 에러율 > 1% -> Slack #alerts

## 핵심 알림 조건

- 웹 앱 다운 -> 즉시 알림
- 트레이딩 엔진 다운 -> 즉시 알림 (시장 시간 중 CRITICAL)
- DB 연결 실패 -> 즉시 알림
- 에러율 급증 (5분 내 10건+) -> 알림
- p95 응답시간 > 2초 -> 경고
