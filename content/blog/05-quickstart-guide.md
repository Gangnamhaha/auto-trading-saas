# AutoTrade KR 시작하기: 5분 퀵스타트 가이드

AutoTrade KR은 한국 주식 자동매매를 위한 노코드 플랫폼입니다. 이 가이드에서는 가입부터 첫 자동매매까지의 전 과정을 단계별로 안내합니다.

## 1단계: 계정 생성

AutoTrade KR 웹사이트(https://autotrade.kr)에 접속하여 회원가입을 진행합니다.

**가입 방법**:

- 이메일로 가입
- Google 계정으로 가입
- 카카오 계정으로 가입

이메일 인증을 완료하면 계정이 활성화됩니다. 무료 플랜으로 시작하며, 언제든 유료 플랜으로 업그레이드할 수 있습니다.

## 2단계: 브로커 연결

자동매매를 위해 증권사 계좌를 연결합니다.

**지원 증권사**:

- 한국투자증권 (KIS)
- 키움증권
- 삼성증권
- 신한투자증권
- (추후 확장 예정)

**연결 방법**:

1. 대시보드에서 "브로커 연결" 메뉴 선택
2. 사용 중인 증권사 선택
3. API 키 입력 (증권사 홈페이지에서 발급)
4. 계좌 번호 입력
5. 연결 테스트 후 완료

API 키는 암호화되어 저장되며, AutoTrade KR은 출금 권한이 없습니다. 매수/매도만 가능합니다.

## 3단계: 전략 선택 또는 생성

AutoTrade KR은 두 가지 방법으로 전략을 설정할 수 있습니다.

**템플릿 전략 사용**:

- 이동평균 크로스오버
- RSI 과매도/과매수
- 볼린저밴드 돌파
- 듀얼 모멘텀
- (추후 확장 예정)

**커스텀 전략 생성**:
전략 빌더를 사용해 드래그 앤 드롭으로 전략을 구성합니다.

**전략 구성 요소**:

- 진입 조건 (매수 신호)
- 청산 조건 (매도 신호)
- 손절매 설정
- 포지션 크기
- 거래 종목 (개별 종목 또는 섹터)

## 4단계: 백테스팅

전략을 실전에 적용하기 전에 백테스팅으로 검증합니다.

**백테스팅 설정**:

1. 기간 선택 (기본: 최근 5년)
2. 초기 자금 설정
3. 거래 비용 반영 (수수료, 거래세)
4. 실행

**결과 확인**:

- 누적 수익률
- 최대 낙폭
- 승률
- 손익비
- 월별/연도별 성과
- 거래 내역 상세

백테스팅 결과가 만족스럽지 않다면 전략 파라미터를 조정합니다. 단, 과적합에 주의하세요.

## 5단계: 페이퍼트레이딩

백테스팅 결과가 좋더라도 페이퍼트레이딩(모의투자)으로 충분히 검증합니다.

**페이퍼트레이딩 특징**:

- 실제 시장 데이터 사용
- 가상 자금으로 거래
- 실전과 동일한 조건
- 리스크 없이 전략 테스트

최소 1개월 이상 페이퍼트레이딩을 진행하세요. 다양한 시장 상황에서 전략이 어떻게 작동하는지 확인합니다.

## 6단계: 실전 투자 시작

페이퍼트레이딩 결과가 만족스러우면 실전 투자를 시작합니다.

**실전 투자 시작 전 체크리스트**:

- [ ] 백테스팅 결과 확인
- [ ] 페이퍼트레이딩 최소 1개월
- [ ] 손절매 설정 완료
- [ ] 일일 손실 한도 설정
- [ ] 소액으로 시작

**권장 사항**:

- 처음에는 총 자금의 10%~20%만 사용
- 한 번의 거래에 총 자금의 2%~5% 투자
- 정기적으로 성과 모니터링
- 문제 발생 시 즉시 전략 중단

## 자주 묻는 질문

**Q: 프로그래밍 지식이 필요한가요?**
A: 아니요. AutoTrade KR은 노코드 플랫폼입니다. 드래그 앤 드롭으로 전략을 구성할 수 있습니다.

**Q: 어떤 증권사를 지원하나요?**
A: 현재 한국투자증권, 키움증권, 삼성증권, 신한투자증권을 지원합니다. 추후 더 많은 증권사를 지원할 예정입니다.

**Q: 출금이 가능한가요?**
A: AutoTrade KR은 출금 권한이 없습니다. 매수/매도만 가능하며, 자금 이체는 증권사 앱에서 직접 하셔야 합니다.

**Q: 수수료는 어떻게 되나요?**
A: Free 플랜은 무료, Basic 플랜은 월 9,900원, Pro 플랜은 월 29,900원입니다. 거래 수수료는 증권사 수수료와 동일합니다.

**Q: 안전한가요?**
A: API 키는 암호화되어 저장되며, 출금 권한이 없습니다. 2단계 인증을 사용하여 계정 보안을 강화할 수 있습니다.
