# AutoTrade KR 커뮤니티 운영 가이드

## 공식 채널

- **카카오톡 오픈채팅:** "AutoTrade KR 베타"
- **디스코드:** discord.gg/autotrading-kr (예정)
- **공식 이메일:** support@autotrade.kr

## 운영 규칙

### 필수 준수 사항

1. 투자 수익률 보장 발언 금지
2. 타 서비스 비방 금지
3. 스팸/광고 금지
4. 개인 투자 결과 공유 시 면책 고지 필수
5. 개인정보(계좌번호, API 키 등) 공유 금지

### 위반 시 조치

- 1회 경고 → 2회 경고 → 채널 퇴장 처리

## Q&A 대응 프로세스

### 기술 문의

- 응답 목표: 24시간 내 답변
- 담당: 운영팀 (기술 지원)
- 복잡한 문의는 이메일로 안내

### 투자 조언 요청 시 대응

- "투자 판단은 본인 책임" 안내 후 기능 설명만 제공
- 수익률 예측, 종목 추천 절대 금지
- 전략 설명 시 반드시 과거 데이터 기반임을 명시

### 장애/오류 신고

- 즉시 확인 후 상태 업데이트 공지
- 복구 예상 시간 안내

## 금지 발언 예시

### ❌ 절대 금지

- "이 전략으로 월 10% 수익 가능합니다"
- "손실 없이 안전하게 투자하세요"
- "이 종목 지금 사면 오릅니다"
- "원금 보장됩니다"
- "수익률 XX% 보장"

### ✅ 올바른 표현

- "이 전략의 2024년 백테스트 결과는 X%였습니다 (과거 결과이며 미래를 보장하지 않음)"
- "페이퍼트레이딩으로 먼저 검증해보시길 권장합니다"
- "투자 결정은 본인의 판단과 책임 하에 이루어져야 합니다"

## 콘텐츠 가이드라인

### 공유 가능한 내용

- 백테스트 결과 (면책 고지 포함)
- 전략 설정 방법 및 팁
- 기능 사용 후기
- 버그 리포트 및 개선 제안

### 공유 불가 내용

- 실제 계좌 수익/손실 (면책 고지 없이)
- 특정 종목 매수/매도 추천
- 타 서비스 비교 비방

## 밋업/웨비나 계획

### 온라인 웨비나 (월 1회)

- 내용: 전략 해설 + 기능 업데이트 + Q&A
- 플랫폼: Zoom 또는 YouTube Live
- 시간: 평일 저녁 8시 (1시간)

### 오프라인 밋업 (분기 1회)

- 장소: 서울 (강남/여의도 예정)
- 내용: 사용자 네트워킹 + 제품 피드백 세션
- 참가비: 무료 (베타 기간)

⚠️ **면책 고지**
AutoTrade KR은 투자자문업자가 아닙니다. 본 커뮤니티에서 공유되는 모든 정보는 참고용이며, 투자 결정의 근거로 사용해서는 안 됩니다. 투자 원금 손실이 발생할 수 있으며, 모든 투자 결정과 그 결과는 투자자 본인의 책임입니다.
