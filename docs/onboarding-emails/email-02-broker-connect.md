# 이메일 02: 브로커 연결 안내 (가입 후 1일 발송)

**제목:** 브로커 연결하셨나요? KIS API 연결 가이드

---

안녕하세요 {이름}님,

아직 브로커를 연결하지 않으셨나요?
브로커 연결이 완료되어야 백테스팅과 페이퍼트레이딩을 시작할 수 있습니다.

**KIS API 연결 3단계:**

**1단계:** KIS Developers 가입

- 접속: apiportal.koreainvestment.com
- 회원가입 후 로그인

**2단계:** 모의투자 App Key/Secret 발급

- [앱 신청] → [모의투자] 선택
- App Key와 App Secret 복사

**3단계:** AutoTrade KR에 입력

- [설정 → 브로커 연결] 메뉴 접속
- App Key / App Secret 입력 후 저장

[브로커 연결 가이드 보기 → /docs/broker-setup]
[지금 연결하기 → /settings/broker]

연결에 어려움이 있으시면 답장으로 문의해 주세요.

AutoTrade KR 팀 드림
