# 이메일 03: 첫 백테스트 안내 (가입 후 3일 발송)

**제목:** 백테스팅으로 전략을 검증해보세요 📊

---

안녕하세요 {이름}님,

실전 투자 전에 반드시 백테스팅을 해보세요!

**백테스팅이란?**
과거 데이터로 전략의 수익성을 시뮬레이션하는 방법입니다.
실제 돈을 쓰지 않고 전략의 성과를 미리 확인할 수 있습니다.

**지금 해보세요 (5분이면 충분합니다):**

1. [백테스트 메뉴 → /backtest] 접속
2. **이동평균 크로스오버 전략** 선택 (초보자 추천)
3. 종목: **삼성전자(005930)** 입력
4. 기간: **2024년 1월 ~ 12월** 설정
5. **백테스트 실행** 클릭 → 결과 확인

**결과에서 확인할 것:**

- 총 수익률 (과거 기준)
- 최대 낙폭 (MDD): 전략의 위험도
- 승률: 수익 거래 비율
- 샤프 비율: 위험 대비 수익

[백테스트 시작하기 → /backtest]
