## 🎬 전체 스크립트

---

### [인트로 — 0:00~0:30]

**[화면: 주식 차트 + AutoTrade KR 로고 애니메이션]**

안녕하세요, AutoTrade KR입니다!

혹시 이런 생각 해보신 적 있으신가요?

_"주식 자동매매 해보고 싶은데... 코딩은 모르고, 어디서 시작해야 할지 모르겠다."_

오늘 이 영상 하나로 해결해 드리겠습니다.

**[화면: 타이머 5:00 표시]**

지금부터 딱 5분 안에, AutoTrade KR로 주식 자동매매를 시작하는 방법을 알려드릴게요.

회원가입부터 첫 번째 페이퍼트레이딩 시작까지, 코딩 한 줄 없이 가능합니다.

자, 시작해볼까요?

---

### [면책 고지 — 0:30~0:45]

**[화면: 주의 아이콘 + 텍스트 오버레이]**

시작 전에 중요한 안내 말씀 드리겠습니다.

주식 투자는 **원금 손실이 발생할 수 있습니다.**

이 영상은 AutoTrade KR 서비스 사용 방법을 안내하는 것이며, **투자 조언이 아닙니다.**

과거 백테스트 성과는 미래 수익을 보장하지 않습니다.

모든 투자 결정은 본인의 판단과 책임 하에 이루어져야 합니다.

---

### [1단계: 회원가입 — 0:45~1:15]

**[화면: AutoTrade KR 웹사이트 화면 캡처]**

첫 번째 단계, 회원가입입니다.

AutoTrade KR 웹사이트에 접속해서 오른쪽 상단의 **"무료 시작하기"** 버튼을 클릭합니다.

**[화면: 회원가입 폼]**

이메일 주소와 비밀번호를 입력하고, 이용약관에 동의하면 끝입니다.

이메일 인증 링크를 클릭하면 바로 로그인이 가능합니다.

**[화면: 대시보드 화면]**

짜잔! 이게 AutoTrade KR 대시보드입니다.

처음 보시면 복잡해 보일 수 있는데, 걱정 마세요. 지금부터 하나씩 설명해 드릴게요.

---

### [2단계: KIS API 연결 — 1:15~2:15]

**[화면: 설정 메뉴 → API 연동]**

두 번째 단계, 증권사 API 연결입니다.

AutoTrade KR은 현재 **한국투자증권(KIS) API**를 지원합니다.

**[화면: 한국투자증권 홈페이지]**

한국투자증권 홈페이지에서 API 신청을 먼저 해야 합니다.

로그인 후 → 마이페이지 → Open API → 신청하기

신청 후 **앱 키(App Key)**와 **앱 시크릿(App Secret)**을 발급받으세요.

**[화면: AutoTrade KR API 설정 화면]**

다시 AutoTrade KR로 돌아와서, 설정 → API 연동 메뉴로 이동합니다.

발급받은 앱 키와 앱 시크릿을 입력하고 **"연결 테스트"** 버튼을 클릭합니다.

**[화면: 연결 성공 메시지]**

초록색 체크 표시가 뜨면 연결 성공입니다!

이제 실제 계좌 정보가 연동되었습니다.

> **💡 팁**: 처음에는 모의투자 계좌로 연결하는 것을 강력히 권장합니다.

---

### [3단계: 전략 선택 — 2:15~2:45]

**[화면: 전략 라이브러리 화면]**

세 번째 단계, 전략 선택입니다.

왼쪽 메뉴에서 **"전략 라이브러리"**를 클릭합니다.

다양한 전략 템플릿이 있는데요, 오늘은 가장 기본적인 **이동평균 크로스오버 전략**을 선택해 보겠습니다.

**[화면: 전략 상세 화면]**

전략 카드를 클릭하면 전략 설명, 파라미터 설정, 과거 성과 요약을 볼 수 있습니다.

**"이 전략 사용하기"** 버튼을 클릭합니다.

---

### [4단계: 백테스트 실행 — 2:45~3:45]

**[화면: 백테스트 설정 화면]**

네 번째 단계, 백테스트입니다.

백테스트란 과거 데이터로 전략을 시뮬레이션해보는 것입니다.

설정 화면에서:

- **종목**: 삼성전자 (005930)
- **기간**: 최근 3년 (2023~2025)
- **단기 이동평균**: 5일
- **장기 이동평균**: 20일
- **초기 자본**: 1,000만원

**[화면: 백테스트 실행 중 로딩]**

**"백테스트 실행"** 버튼을 클릭합니다.

약 10초 후 결과가 나옵니다.

**[화면: 백테스트 결과 화면]**

결과 화면에서 볼 수 있는 주요 지표들:

- **총 수익률**: 전략의 전체 성과
- **MDD (최대 낙폭)**: 최악의 경우 얼마나 손실이 났는지
- **샤프비율**: 리스크 대비 수익 효율성
- **승률**: 매매 중 수익이 난 비율

> ⚠️ **중요**: 이 결과는 과거 데이터 기반이며, 미래 수익을 보장하지 않습니다.

---

### [5단계: 페이퍼트레이딩 시작 — 3:45~4:45]

**[화면: 페이퍼트레이딩 설정 화면]**

다섯 번째 단계, 페이퍼트레이딩입니다.

페이퍼트레이딩은 **실제 돈 없이** 자동매매를 연습하는 방법입니다.

백테스트 결과 화면에서 **"페이퍼트레이딩 시작"** 버튼을 클릭합니다.

**[화면: 페이퍼트레이딩 설정]**

설정:

- **모의 자본**: 1,000만원
- **1회 매매 금액**: 100만원 (자본의 10%)
- **손절매**: -5%
- **익절매**: +10%

**"시작"** 버튼을 클릭합니다.

**[화면: 페이퍼트레이딩 대시보드]**

이제 실시간으로 전략이 작동합니다!

매수/매도 시그널이 발생하면 알림을 받을 수 있고, 대시보드에서 성과를 실시간으로 확인할 수 있습니다.

> 💡 **권장**: 최소 1~3개월 페이퍼트레이딩 후 실전 투자를 고려하세요.

---

**[화면: AutoTrade KR 로고 + 링크]**

어떠셨나요? 생각보다 간단하죠?

설명란에 **무료 체험 링크**를 남겨두었습니다.

지금 바로 시작해보세요!
