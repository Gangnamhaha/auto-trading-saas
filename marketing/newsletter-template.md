## 📋 뉴스레터 작성 가이드라인

### 제목 작성 원칙

1. **숫자 포함**: "KOSPI 2,700 돌파" vs "KOSPI 상승"
2. **긴급성/관련성**: 이번 주 시장 이슈와 연결
3. **구체성**: 모호한 표현 지양
4. **50자 이내**: 모바일에서 잘리지 않도록

### 콘텐츠 작성 원칙

1. **간결함**: 각 섹션 3~5줄 이내
2. **데이터 기반**: 수치와 사실 중심
3. **면책 고지**: 투자 관련 내용마다 포함
4. **CTA 명확**: 각 섹션에 하나의 행동 유도

### 발행 전 체크리스트

- [ ] 시장 데이터 최신 업데이트 확인
- [ ] 전략 성과 수치 정확성 확인
- [ ] 면책 고지 모든 섹션 포함 확인
- [ ] 링크 작동 확인
- [ ] 수신 거부 링크 작동 확인
- [ ] 모바일 미리보기 확인
- [ ] 스팸 필터 테스트
