# AutoTrade KR — 12개월 광고 예산 계획

> 최종 업데이트: 2026-03-15  
> 총 예산: 약 5,000-7,000만원 (12개월)

---

## 예산 전략 개요

광고 예산은 3단계 Phase로 운영한다. 초기에는 소규모 테스트로 채널별 CAC를 측정하고, 성과가 검증된 채널에 집중 투자하여 효율을 극대화한다.

```
Phase 1 (M1-3): 테스트    → 월 100만원  → CAC 측정
Phase 2 (M4-6): 최적화   → 월 300만원  → CAC 4만원 이하 달성
Phase 3 (M7-12): 스케일업 → 월 500-1,000만원 → ROAS 3:1 달성
```

### Phase 1 상세 계획

- Google Ads: 검색 캠페인 세팅, 핵심 키워드 10개 테스트
- Meta Ads: 관심사 타겟 3개 세트 테스트
- YouTube: 30초 데모 영상 1개 제작 및 집행
- 네이버: 파워링크 기본 세팅

**Month 2: 데이터 수집**

- 채널별 CTR, CVR, CAC 데이터 수집
- 성과 낮은 키워드/타겟 제거
- 성과 좋은 크리에이티브 예산 증액

**Month 3: 최적화 준비**

- Phase 1 성과 분석 보고서 작성
- Phase 2 예산 배분 결정
- 리타겟팅 오디언스 구축 (최소 1,000명)

## 예산 집행 원칙

### 증액 기준 (다음 Phase 진입 조건)

- [ ] CAC가 목표치(4만원) 이하로 2주 연속 유지
- [ ] ROAS 2:1 이상 달성
- [ ] 전환 추적 데이터 신뢰도 확보 (최소 50건 전환)

### 감액/중단 기준

- CAC가 목표치 2배(8만원) 초과 시 해당 채널 예산 50% 감액
- ROAS 0.5:1 이하 2주 지속 시 캠페인 일시 중지
- 클릭 사기(Click Fraud) 의심 시 즉시 중단 및 조사

### 예비 예산

- 각 Phase 예산의 10%를 예비비로 확보
- 갑작스러운 경쟁 심화 또는 시즌 기회 대응

### 월별 리뷰 프로세스

1. 매월 1일: 전월 성과 분석 보고서 작성
2. 매월 3일: 채널별 예산 조정 결정
3. 매월 5일: 새 예산으로 캠페인 업데이트
4. 매주 월요일: 주간 성과 모니터링
