# AutoTrade KR — 유료 광고 채널 전략서

> 최종 업데이트: 2026-03-15  
> 법적 구조: 비자문형 거래자동화 도구 — 모든 광고에 면책 고지 필수

---

## 개요

AutoTrade KR의 유료 광고는 4개 채널을 통해 운영된다. 각 채널은 서로 다른 사용자 접점을 커버하며, 단계별 예산 배분을 통해 CAC 4만원 이하를 목표로 한다.

| 채널             | 특성           | 주요 목표                 |
| ---------------- | -------------- | ------------------------- |
| Google Ads       | 검색 의도 기반 | 고의도 사용자 포착        |
| Meta Ads         | 관심사 기반    | 잠재 고객 발굴 + 리타겟팅 |
| YouTube Ads      | 영상 콘텐츠    | 브랜드 인지도 + 데모      |
| 네이버 검색 광고 | 한국 검색 특화 | 국내 검색 시장 커버       |

---

## 1. Google Ads (검색 광고)

### 전략 개요

검색 의도가 명확한 사용자를 포착하는 가장 직접적인 채널. "주식 자동매매"를 검색하는 사용자는 이미 솔루션을 찾고 있는 상태이므로 전환율이 높다.

### 타겟 키워드

**핵심 키워드 (High Intent)**
| 키워드 | 예상 월 검색량 | 예상 CPC |
|--------|--------------|---------|
| 주식 자동매매 | 5,000-10,000 | 1,000-2,000원 |
| 자동매매 프로그램 | 3,000-7,000 | 800-1,500원 |
| 퀀트 투자 프로그램 | 1,000-3,000 | 500-1,200원 |
| 주식 자동매매 프로그램 | 2,000-5,000 | 700-1,500원 |

**롱테일 키워드 (Lower CPC)**
| 키워드 | 예상 CPC |
|--------|---------|
| 코딩없이 자동매매 | 300-700원 |
| 백테스팅 프로그램 | 400-900원 |
| 한국 주식 자동매매 | 500-1,000원 |
| 퀀트 투자 입문 | 300-600원 |

**제외 키워드**

- 무료 자동매매 (구매 의도 낮음)
- 자동매매 사기 (부정적 연관)
- 해외 주식 자동매매 (서비스 범위 외)

### 광고 형식

- **형식**: 반응형 검색 광고 (RSA)
- **헤드라인**: 최대 15개 준비, 자동 최적화
- **설명**: 최대 4개, 핵심 가치 제안 포함
- **광고 확장**: 사이트링크, 콜아웃, 구조화된 스니펫

### 광고 문안 예시

```
헤드라인 후보:
- 한국 주식 자동매매 플랫폼
- 코딩 없이 퀀트 투자 시작
- 백테스팅으로 전략 검증 후 투자
- 30일 무료 체험 | AutoTrade KR
- 무료 가입 후 즉시 시작

설명 후보:
- 코딩 없이 나만의 자동매매 전략을 만들고 백테스팅으로 검증하세요. 30일 페이퍼트레이딩 무료.
- 투자 원금 손실 가능. 과거 수익률이 미래를 보장하지 않음. 무료로 시작해 전략을 검증하세요.
```

### 랜딩페이지

- **URL**: autotrading.kr (메인 랜딩)
- **전환 목표**: 무료 가입 (이메일 등록)
- **추적**: Google Ads 전환 태그 + GA4 연동

### Phase별 예산 및 목표

| Phase           | 월 예산     | 예상 클릭   | 예상 전환율 | 예상 신규 가입 | 예상 CAC        |
| --------------- | ----------- | ----------- | ----------- | -------------- | --------------- |
| Phase 1 (M1-3)  | 40만원      | 300-500     | 3-5%        | 10-25명        | 16,000-40,000원 |
| Phase 2 (M4-6)  | 120만원     | 900-1,500   | 4-6%        | 36-90명        | 13,000-33,000원 |
| Phase 3 (M7-12) | 200-400만원 | 1,500-4,000 | 5-7%        | 75-280명       | 14,000-27,000원 |

### 최적화 전략

1. **Quality Score 개선**: 키워드-광고-랜딩페이지 연관성 최대화
2. **입찰 전략**: 전환 최대화 → 목표 CPA 설정 (4만원)
3. **시간대 조정**: 평일 오전 9-11시, 오후 7-10시 입찰가 상향
4. **기기 조정**: 모바일 vs 데스크탑 전환율 비교 후 조정
5. **부정 키워드 지속 추가**: 검색어 보고서 주 1회 검토

## 2. Meta Ads (Instagram / Facebook)

### 전략 개요

관심사 기반 타겟팅으로 잠재 고객을 발굴하고, 랜딩페이지 방문자를 리타겟팅하는 이중 전략. 시각적 콘텐츠(백테스트 결과 차트)가 강점.

### 타겟팅 설정

**기본 타겟**

- 연령: 25-45세
- 지역: 대한민국
- 언어: 한국어

**관심사 타겟**

- 주식 투자, 재테크, 부동산 투자
- 경제/금융 뉴스 (한국경제, 매일경제)
- 핀테크 앱 (토스, 카카오페이)
- 퀀트 투자, 알고리즘 트레이딩

**행동 타겟**

- 금융 앱 사용자
- 온라인 쇼핑 구매자 (결제 의향 있음)
- 소규모 사업자 (투자 여력 있음)

**리타겟팅 (핵심)**

- autotrading.kr 방문자 (30일 이내)
- 가입 페이지 도달 후 미완료자
- 영상 광고 25% 이상 시청자
- 기존 무료 사용자 (유료 전환 유도)

**유사 타겟 (Lookalike)**

- 유료 전환 사용자 기반 1-3% 유사 타겟
- 이메일 가입자 기반 유사 타겟

### 광고 형식

**이미지 광고**

- 백테스트 결과 스크린샷 (수익 곡선 차트)
- 대시보드 UI 스크린샷
- 인포그래픽 (자동매매 작동 원리)
- 사이즈: 1080×1080 (피드), 1080×1920 (스토리)

**카루셀 광고**

- 슬라이드 1: 문제 제기 ("매일 주식 보느라 지치셨나요?")
- 슬라이드 2: 솔루션 소개 (AutoTrade KR)
- 슬라이드 3: 백테스트 결과 시각화
- 슬라이드 4: 가격 및 CTA

**동영상 광고**

- 15초: 핵심 가치 제안
- 30초: 플랫폼 데모

### Phase별 예산 및 목표

| Phase           | 월 예산     | 예상 CPM | 예상 도달         | 예상 전환율 | 예상 CAC        |
| --------------- | ----------- | -------- | ----------------- | ----------- | --------------- |
| Phase 1 (M1-3)  | 30만원      | 8,000원  | 37,500명          | 0.5-1%      | 30,000-60,000원 |
| Phase 2 (M4-6)  | 90만원      | 7,000원  | 128,571명         | 1-2%        | 15,000-30,000원 |
| Phase 3 (M7-12) | 150-300만원 | 6,000원  | 250,000-500,000명 | 1.5-2.5%    | 10,000-20,000원 |

\*리타겟팅 캠페인은 신규 캠페인 대비 전환율 3-5배 높음

### 최적화 전략

1. **캠페인 구조**: 인지(Awareness) → 고려(Consideration) → 전환(Conversion) 퍼널
2. **크리에이티브 로테이션**: 2주마다 새 크리에이티브 테스트
3. **픽셀 최적화**: Meta Pixel 설치 → 전환 이벤트 최적화
4. **예산 배분**: 신규 타겟 70% / 리타겟팅 30%
5. **광고 피로도 관리**: 빈도(Frequency) 3 이상 시 크리에이티브 교체

## 3. YouTube Ads (인스트림)

### 전략 개요

투자/재테크 채널 시청자에게 플랫폼 데모 영상을 노출. 브랜드 인지도 구축과 함께 실제 사용 경험을 시각적으로 전달.

### 타겟팅 설정

**채널/콘텐츠 타겟**

- 주식 투자 유튜브 채널 (슈카월드, 삼프로TV 등)
- 재테크/경제 채널
- 퀀트 투자 관련 채널
- 핀테크 리뷰 채널

**관심사 타겟**

- 금융 및 투자
- 주식 시장
- 개인 금융 관리

**키워드 타겟**

- "주식 투자", "자동매매", "퀀트 투자", "재테크"

**리마케팅**

- Google Ads 리마케팅 목록 연동
- YouTube 채널 방문자

### 광고 형식

**5초 스킵 가능 인스트림 (TrueView)**

- 처음 5초에 핵심 메시지 전달
- 스킵 후에도 브랜드 노출 효과
- 과금: 30초 이상 시청 또는 클릭 시

**범퍼 광고 (6초, 스킵 불가)**

- 브랜드 인지도 캠페인용
- 짧고 강렬한 메시지

### 크리에이티브 구성 (30초 데모 영상)

```
[0-5초] 훅 (스킵 전 필수 전달)
"주식 자동매매, 코딩 없이 가능합니다"
→ 플랫폼 대시보드 화면 등장

[5-15초] 문제-솔루션
"매일 주식 차트 보느라 지치셨나요?
AutoTrade KR로 전략을 설정하면 자동으로 매매합니다"
→ 전략 설정 화면 시연

[15-25초] 신뢰 구축
"백테스팅으로 실제 투자 전 전략을 검증하세요"
→ 백테스트 결과 차트 표시

[25-30초] CTA
"지금 무료로 시작하세요 — autotrading.kr"
→ 면책 고지: "투자 원금 손실 가능. 과거 수익률이 미래를 보장하지 않음."
```

### Phase별 예산 및 목표

| Phase           | 월 예산     | 예상 CPV | 예상 조회수     | 예상 전환율 | 예상 CAC         |
| --------------- | ----------- | -------- | --------------- | ----------- | ---------------- |
| Phase 1 (M1-3)  | 20만원      | 100원    | 2,000회         | 1-2%        | 50,000-100,000원 |
| Phase 2 (M4-6)  | 60만원      | 80원     | 7,500회         | 1.5-3%      | 20,000-40,000원  |
| Phase 3 (M7-12) | 100-200만원 | 70원     | 14,000-28,000회 | 2-4%        | 18,000-35,000원  |

\*YouTube는 직접 전환보다 브랜드 인지도 → 검색 전환 경로가 많음

### 최적화 전략

1. **조회율(VTR) 목표**: 30% 이상 유지
2. **채널 제외**: 어린이 콘텐츠, 게임, 관련 없는 채널 제외
3. **시퀀스 광고**: 인지 → 고려 → 전환 순서로 노출
4. **브랜드 리프트 측정**: 인지도 변화 추적
5. **영상 A/B 테스트**: 훅 문구 2-3가지 테스트

## 4. 네이버 검색 광고 (파워링크)

### 전략 개요

한국 검색 시장의 약 60%를 점유하는 네이버에서 검색 광고를 운영. Google Ads와 상호 보완적으로 운영하여 검색 커버리지를 극대화.

### 타겟 키워드

**핵심 키워드**
| 키워드 | 예상 월 검색량 | 예상 CPC |
|--------|--------------|---------|
| 주식 자동매매 | 8,000-15,000 | 800-1,500원 |
| 자동매매 프로그램 | 5,000-10,000 | 600-1,200원 |
| 주식 자동매매 프로그램 | 3,000-7,000 | 700-1,300원 |
| 퀀트 투자 | 2,000-5,000 | 300-700원 |

**롱테일 키워드**
| 키워드 | 예상 CPC |
|--------|---------|
| 주식 자동매매 추천 | 400-900원 |
| 자동매매 백테스팅 | 300-600원 |
| 코딩없이 자동매매 | 300-500원 |

### 광고 형식

- **파워링크**: 검색 결과 상단 텍스트 광고
- **브랜드 검색**: "AutoTrade KR" 검색 시 브랜드 영역 확보
- **쇼핑 검색**: 해당 없음 (서비스 상품)

### 광고 문안 예시

```
제목: 한국 주식 자동매매 | AutoTrade KR
설명: 코딩 없이 나만의 자동매매 전략을 만들고 백테스팅으로 검증하세요.
      무료 가입 후 30일 페이퍼트레이딩 체험 가능.
      투자 원금 손실 가능. 과거 수익률이 미래를 보장하지 않음.
URL: autotrading.kr
```

### Phase별 예산 및 목표

| Phase           | 월 예산    | 예상 클릭 | 예상 전환율 | 예상 신규 가입 | 예상 CAC        |
| --------------- | ---------- | --------- | ----------- | -------------- | --------------- |
| Phase 1 (M1-3)  | 10만원     | 100-200   | 3-5%        | 3-10명         | 10,000-33,000원 |
| Phase 2 (M4-6)  | 30만원     | 300-600   | 4-6%        | 12-36명        | 8,000-25,000원  |
| Phase 3 (M7-12) | 50-100만원 | 500-1,500 | 5-7%        | 25-105명       | 5,000-20,000원  |

### 최적화 전략

1. **품질지수 관리**: 클릭률 + 랜딩페이지 연관성 최적화
2. **입찰가 조정**: 시간대별, 기기별 성과 분석 후 조정
3. **네이버 애널리틱스 연동**: 전환 추적 설정
4. **콘텐츠 광고 병행**: 네이버 블로그/카페 콘텐츠 마케팅과 시너지
5. **브랜드 검색 광고**: 경쟁사가 브랜드명 키워드 입찰 방어

## 채널 통합 전략

### 멀티채널 퍼널 설계

```
[인지 단계]
YouTube 인스트림 → 브랜드 인지도 형성
Meta 피드 광고 → 관심사 기반 노출

[고려 단계]
Google/네이버 검색 → 능동적 검색 포착
Meta 리타겟팅 → 방문자 재접촉

[전환 단계]
검색 광고 → 직접 전환
Meta 리타겟팅 → 전환 완료 유도

[유지 단계]
Meta 기존 사용자 → 유료 업그레이드 유도
```

### 채널별 역할 분담

| 채널       | 주요 역할      | 보조 역할        |
| ---------- | -------------- | ---------------- |
| Google Ads | 직접 전환      | 브랜드 검색 방어 |
| Meta Ads   | 잠재 고객 발굴 | 리타겟팅 전환    |
| YouTube    | 브랜드 인지도  | 데모 시연        |
| 네이버     | 국내 검색 커버 | 직접 전환        |

### 통합 추적 설정

- **UTM 파라미터**: 모든 광고 URL에 채널/캠페인/광고그룹 태그
- **GA4 연동**: 모든 채널 전환 데이터 통합
- **어트리뷰션 모델**: 데이터 기반 어트리뷰션 (첫 터치 vs 마지막 터치 비교)
