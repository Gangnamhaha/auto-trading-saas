# AutoTrade KR — 광고 크리에이티브 가이드라인

> 최종 업데이트: 2026-03-15  
> 법적 구조: 비자문형 거래자동화 도구  
> 모든 광고 집행 전 이 가이드라인 준수 여부 확인 필수

---

## 핵심 원칙

AutoTrade KR은 **"비자문형 거래자동화 도구"**다. 투자 조언이나 수익을 보장하는 서비스가 아니다. 모든 광고 크리에이티브는 이 포지셔닝을 명확히 반영해야 한다.

**브랜드 포지셔닝**: 코딩 없이 자동매매 전략을 만들고 백테스팅으로 검증하는 도구

## 필수 면책 고지

### 기본 면책 고지 (모든 광고 필수)

```
투자 원금 손실 가능. 과거 수익률이 미래를 보장하지 않음.
```

### 백테스트 결과 표시 시 추가 고지

```
위 결과는 과거 데이터 기반 백테스팅 결과이며, 실제 투자 성과와 다를 수 있습니다.
투자 원금 손실 가능. 과거 수익률이 미래를 보장하지 않음.
```

### 영상 광고 면책 고지

- 영상 마지막 3초 이상 화면에 표시
- 폰트 크기: 최소 12pt (전체 화면 기준)
- 배경과 충분한 대비 확보

### 이미지 광고 면책 고지

- 이미지 하단에 명확히 표시
- 폰트 크기: 이미지 크기에 비례하여 가독성 확보
- 흐리게 처리하거나 숨기지 않을 것

## 채널별 크리에이티브 가이드

### Google Ads (텍스트 광고)

**헤드라인 작성 원칙**

- 키워드 포함 (검색어와 연관성)
- 핵심 가치 제안 명확히
- 행동 유도 문구 포함

**추천 헤드라인 조합**

```
헤드라인 1: 한국 주식 자동매매 플랫폼
헤드라인 2: 코딩 없이 퀀트 투자 시작
헤드라인 3: 30일 무료 체험 | AutoTrade KR
헤드라인 4: 백테스팅으로 전략 검증
헤드라인 5: 무료 가입 후 즉시 시작
```

**설명 문구 원칙**

- 2개 이상 준비
- 면책 고지 포함 (축약 가능)
- CTA 명확히

```
설명 예시 1:
코딩 없이 나만의 자동매매 전략을 만들고 백테스팅으로 검증하세요. 무료 가입 가능.

설명 예시 2:
30일 페이퍼트레이딩으로 실전 전 연습. 투자 원금 손실 가능. 과거 수익률 미래 미보장.
```

---

### Meta Ads (이미지/카루셀)

**이미지 광고 구성**

```
[상단 1/3] 핵심 메시지 텍스트
  - 폰트: 굵고 읽기 쉬운 한글 폰트
  - 배경: 브랜드 컬러 (진한 색상)
  - 예: "코딩 없이 자동매매 시작"

[중간 1/3] 플랫폼 UI 스크린샷 또는 차트
  - 실제 플랫폼 화면 사용
  - 백테스트 결과 차트 (면책 고지 포함)

[하단 1/3] CTA + 면책 고지
  - CTA 버튼: "무료로 시작하기"
  - 면책 고지: 소형 텍스트로 포함
```

**카루셀 광고 구성 (5장)**

```
슬라이드 1: 문제 제기
  - 이미지: 복잡한 주식 차트 보는 사람
  - 텍스트: "매일 주식 차트 보느라 지치셨나요?"

슬라이드 2: 솔루션 소개
  - 이미지: AutoTrade KR 로고 + 대시보드
  - 텍스트: "AutoTrade KR로 자동화하세요"

슬라이드 3: 핵심 기능 1 (백테스팅)
  - 이미지: 백테스트 결과 차트
  - 텍스트: "투자 전 전략을 데이터로 검증"
  - 면책 고지 포함

슬라이드 4: 핵심 기능 2 (자동 실행)
  - 이미지: 자동 매매 실행 화면
  - 텍스트: "설정 후 자동으로 실행"

슬라이드 5: CTA
  - 이미지: 가격 플랜 요약
  - 텍스트: "지금 무료로 시작하기"
  - CTA 버튼: "무료 가입"
```

**색상 가이드**

- 주 색상: 브랜드 컬러 (진한 파란색 계열 권장)
- 강조 색상: 초록색 (성장/긍정 연상)
- 배경: 흰색 또는 진한 색상 (가독성)
- 금지: 빨간색 단독 사용 (손실 연상)

### YouTube Ads (영상)

**5초 훅 (스킵 전 필수 전달)**

```
옵션 A: 질문형
"주식 자동매매, 코딩 없이 가능하다는 거 알고 계셨나요?"

옵션 B: 문제 제기형
"매일 주식 차트 보느라 지치셨나요? 이제 자동화하세요."

옵션 C: 결과 제시형
"이 전략, 백테스팅으로 먼저 검증했습니다." (면책 고지 포함)
```

**영상 구성 원칙**

- 0-5초: 훅 (스킵 전 핵심 메시지)
- 5-20초: 문제-솔루션 전개
- 20-25초: 신뢰 구축 (백테스팅, 기능 시연)
- 25-30초: CTA + 면책 고지

**영상 금지 사항**

- 수익률 수치 강조 (면책 고지 없이)
- 과도한 흥분/과장 표현
- 빠른 화면 전환으로 면책 고지 가리기

## 크리에이티브 검수 체크리스트

광고 집행 전 아래 항목을 모두 확인한다:

### 법적 준수

- [ ] 수익 보장 문구 없음
- [ ] 경쟁사 비교 없음
- [ ] 면책 고지 포함 (모든 광고)
- [ ] 백테스트 결과 표시 시 추가 고지 포함
- [ ] 과장/허위 표현 없음

### 브랜드 일관성

- [ ] 브랜드 컬러 사용
- [ ] 로고 올바르게 사용
- [ ] 폰트 가이드 준수
- [ ] 서비스명 "AutoTrade KR" 정확히 표기

### 기술적 요건

- [ ] 이미지 해상도 적합 (채널별 권장 사이즈)
- [ ] 텍스트 비율 준수 (Meta: 20% 이하)
- [ ] 랜딩페이지 URL 정상 작동
- [ ] UTM 파라미터 설정 완료
- [ ] 전환 추적 태그 작동 확인

### 품질 기준

- [ ] 면책 고지 가독성 확보
- [ ] CTA 명확히 표시
- [ ] 모바일 화면에서 가독성 확인
- [ ] 오탈자 없음

## 크리에이티브 성과 기준

### 채널별 목표 지표

| 채널       | 지표        | 목표     |
| ---------- | ----------- | -------- |
| Google Ads | CTR         | 3% 이상  |
| Meta Ads   | CTR         | 1% 이상  |
| Meta Ads   | 관련성 점수 | 7 이상   |
| YouTube    | 조회율(VTR) | 30% 이상 |
| 네이버     | CTR         | 2% 이상  |

### 크리에이티브 교체 기준

- CTR이 목표의 50% 이하로 2주 지속 시 교체
- Meta 빈도(Frequency) 3 초과 시 교체
- YouTube 조회율 20% 이하 시 교체

## 승인 프로세스

1. **초안 작성** → 마케팅 담당자
2. **법적 검토** → 면책 고지 포함 여부, 금지 문구 확인
3. **브랜드 검토** → 브랜드 가이드라인 준수 확인
4. **최종 승인** → 팀 리드 승인
5. **집행** → 광고 플랫폼 업로드

_긴급 집행 시에도 법적 검토(2단계)는 반드시 거칠 것_
